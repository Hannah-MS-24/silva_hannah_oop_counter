# Counter Component (JavaScript OOP)

A responsive counter component built with JavaScript using Object-Oriented Programming. This project allows users to increment, decrement, and reset values, with dynamic UI updates and support for multiple instances.

# About 🧩

This project is a reusable counter component designed using JavaScript OOP principles. It provides a simple and interactive way to manage numeric values through a clean UI.

The goal of this assignment was to practice encapsulation, DOM manipulation inside classes, state-driven UI updates, and inheritance by extending the base counter into a more advanced version.

# Features 🌟

📌 Increment, Decrement, and Reset buttons

📌 Prevents values from going below 0

📌 Reset button logs "Reset Activated!!!" in console

📌 Dynamic UI (buttons become inactive at 0)

📌 Custom initial value

📌 Multiple counters on the same page

📌 StepCounter class (advanced version):

Uses inheritance (extends)

Custom step value

Overrides increment and decrement methods

📌 Responsive layout for different screen sizes
