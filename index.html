Reusable Counter Component

A lightweight, modular counter built with JavaScript OOP — featuring encapsulation, state-driven UI, and inheritance via StepCounter.

Overview

What it does

The Counter class is a reusable JavaScript component that creates interactive counters with increment, decrement, and reset functionality. It manages its own state and updates the DOM automatically.

What problem it solves

Instead of rewriting counter logic for every use case, this class provides a self-contained component. Just pass a selector and an optional starting value — the counter handles everything else.

How To Use It

1. Create a container in your HTML

<div id="my-counter"></div>

2. Import the Counter class

import { Counter } from "./js/counter.js";

3. Instantiate the counter

const counter = new Counter("#my-counter", 10);
// selector: required — the container element
// initialValue: optional — defaults to 0

Parameters

  • selector — CSS selector for the HTML container
  • initialValue — Starting value for the counter (default: 0)

Behaviour

  • Counter never goes below 0
  • Decrement and Reset buttons become visually inactive at 0
  • Reset logs Reset Activated!!! to the console
  • Multiple counters can run independently on the same page

StepCounter

StepCounter extends Counter and demonstrates inheritance and polymorphism. It accepts a third parameter — step — and overrides increment() and decrement() to move by that amount instead of 1.

Import StepCounter

import { Counter, StepCounter } from "./js/counter.js";

Instantiate with a step value

const stepCounter = new StepCounter("#my-step-counter", 0, 5);
// selector: required
// initialValue: optional — defaults to 0
// step: optional — defaults to 1

Parameters

  • selector — CSS selector for the HTML container
  • initialValue — Starting value (default: 0)
  • step — Amount to increment/decrement by (default: 1)

OOP Concepts Used

  • Uses extends to inherit from Counter
  • Uses super() to call the parent constructor
  • Overrides increment() and decrement()
  • Overrides update() to display the step value
