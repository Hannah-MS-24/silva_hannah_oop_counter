Reusable Counter Component

A lightweight JavaScript counter built using Object-Oriented Programming, state-driven UI updates, and modular JavaScript.

Overview

The Counter class is a reusable JavaScript component that creates interactive counters that can increment and decrement values.

It solves the problem of rewriting the same counter logic by providing a self-contained class that manages its own state and updates the DOM automatically when the value changes.

How To Use It

1. Create a container in HTML

<div id="counter"></div>

2. Import the module

import Counter from "./js/counter.js";

3. Instantiate the counter

const counter = new Counter("#counter", 10);
counter.mount();

Parameters

  • container – The selector for the HTML container
  • initialValue – The starting value for the counter
